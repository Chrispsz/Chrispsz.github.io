Como Usar

                ×
            

            

                

                    
1. Envie sua Planilha

                    
                    
Arraste e solte um arquivo (`.xlsx`) na área indicada, ou clique para selecioná-lo do seu computador.

                

                

                    
2. Configure as Cores

                    
                    
Use o ícone de engrenagem para abrir as configurações. Lá, você pode criar regras para associar cada cor a uma descrição (ex: vermelho = "pendente").

                

                 

                    
3. Gere e Exporte

                    
                    
A ferramenta irá gerar uma lista formatada automaticamente. Use os botões para copiar a lista ou baixar um arquivo de texto.
